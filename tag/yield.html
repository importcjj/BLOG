Python yield小记

yeild

yield 一般搭配函数来定义一个Generator(生成器)

一个简单的例子:

def f():
    print "Today is 7.21"
    yield 6

f() 将会返回一个Generator, 而非像普通函数一样执行。想要使用生成器的话(比如 i)需要使用i.next() (与next(i)等效) 和i.send(value)。

>>> i = f()
>>> i
<generator object f at 0x1041e7dc0>
>>> i.next()
Today is 7.21
6
>>> i.next()
